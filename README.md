This repository contains a talk given at the [Ticer Summer School 2012][ticer] 
in Tilburg at August 22th.

## SYNOPSIS

Slides are given in Pandoc Markdown syntax in `slides.md`. There is
a `Makefile` to create a PDF:

    make pdf
